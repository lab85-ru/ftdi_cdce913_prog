Программатор TI PLL CDCE913 при помощи FTDI FT232H (или подобной).

Порядок работы.
1. Создать прошивку при помощи ПО TI ClockPro(scac119e.zip).
   В формате BIN (будет файл длинной 32 байта)
2. Подключить FTDI к CDCE913 (gnd, sda, scl).
3. Запустить програматор со следующими параметрами:

   ftdi_cdce913_prog.exe --filecfg FILE.bin --write-reg --eeprom
